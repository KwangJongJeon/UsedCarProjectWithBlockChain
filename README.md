# UsedCarProjectWithBlockChainClient

1. 사용한 프레임워크
<ul>
  <li>TRUFFLE</li>
  <li>REACT</li>
  <li>WEB3<li>
</ul>
<br/><hr/>
2. 설치 필요
<ul>
  <li>Truffle v5.1.8</li>
  <li>chocolatey</li>
  <li>METAMASK </li>
</ul>
<br/><hr/>
3. 커맨드
<br/>
$ npm install <br/>
$ npm install -g truffle@v5.1.8 <br/>

<br/><hr/>
4. Truffle 구동 명령어 <br/>
파일 client 폴더에서 => <br/>
$ truffle development  <br/>
$ (truffleConsole) compile <br/>
$ (truffleConsole) migrate <br/>
$ (truffleConsole) deploy <br/>
<br/><hr/>
5. 서버 구동 <br/>
(client 폴더 위치에서) <br/>
npm start <br/>
